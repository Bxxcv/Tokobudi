FAQ

Pertanyaan yang sering ditanyakan

Ada yang kurang jelas? Cek jawaban di bawah atau hubungi admin langsung.

Apakah ada biaya pendaftaran?
Tidak ada biaya pendaftaran. Cukup hubungi admin via WhatsApp, dan akun toko kamu langsung dibuatkan secara gratis. Paket Basic mulai Rp50.000/bulan sudah bisa langsung aktif.
Berapa lama proses aktivasi akun?
Biasanya kurang dari 5 menit setelah admin menerima data kamu. Kami aktif setiap hari dan merespons dengan cepat.
Apakah saya bisa punya banyak produk?
Ya! Tidak ada batasan jumlah produk. Kamu bisa upload produk sebanyak-banyaknya, lengkap dengan foto, harga, stok, kategori, dan deskripsi.
Apa bedanya paket Basic dan Premium?
Basic sudah cukup untuk mulai berjualan: halaman toko, produk, dan link ke semua platform. Premium menambahkan: analitik real-time, 6 tema eksklusif, badge terverifikasi, QR Code, tombol kustom, dan hapus branding LINKify.
Bisakah saya upgrade/downgrade kapan saja?
Tentu saja. Hubungi admin kapan saja untuk upgrade ke Premium atau jika ingin kembali ke Basic. Tidak ada kontrak jangka panjang.
Data toko saya aman?
Sangat aman. Semua data tersimpan di Firebase (Google Cloud), diproteksi enkripsi SSL, dan hanya bisa diakses oleh akun kamu. Kami tidak pernah menjual atau membagikan data toko ke pihak ketiga.
